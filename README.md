# IPv4Edit

The IPv4Edit Widget Library
----------------------

       You can get IPv4Edit at https://github.com/lizhenneng/IPv4Edit.git

Installation
------------
First:
	qmake IPv4Edit.pro
Second:
	make 
Third:
	cp -rf libipv4editplugin.so /usr/lib/x86_64-linux-gnu/qt4/plugins/designer/
